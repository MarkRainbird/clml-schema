Crown Legislation Markup Language Reference Guide

Welcome

Welcome to the Reference Guide for Crown Legislation Markup Language (CLML), the markup language for representing UK legislation. This guide explains how to use the CLML schema when authoring and interpreting CLML documents.

How to use this guide

For an introduction to the CLML schema, you can read the User Guide, which explains the design and implementation details of the schema and gives examples of its use.

Alternatively, you can consult the Reference Guide for a specific module or component of the schema. You can:

  • choose a specific component from the Table of Contents in the left-hand pane,
  • choose Components in the left-hand pane to view components grouped by schema module,
  • choose Resource Hierarchy in the left-hand pane to view schema modules in a hierarchy that shows which modules include or import other modules, or
  • start at the top level of the schema and navigate down through the components.
