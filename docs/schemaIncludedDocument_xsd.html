Annotations

Included documents are documents not coded to the main schema. They are referenced via the Resources section.

Inline content gives the opportunity to provide some explanatory text as to the content.

An included document references a resource and it can, therefore, be anything that can be held in a Resource element. For forms the recommendation would be to use Scalable Vector Graphics (SVG) to mark up.
